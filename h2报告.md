### VSRN 复现命令

```bash
cd /mnt/kxh/smx/homework/h2/VSRN
python train.py \
  --data_path "$DATA_PATH" \
  --data_name f30k_precomp \
  --logger_name runs/flickr_VSRN \
  --max_violation \
  --lr_update 10 \
  --max_len 60
```

其中 `$DATA_PATH` 应指向 SCAN `data.zip` 解压后的目录。

### 加分项模型

加分项选择 CLIP `openai/clip-vit-base-patch32`。CLIP 是大规模图文对比学习模型，
不需要在 Flickr30k 上重新训练即可进行零样本图文检索。评测时对每张图片和其 5 条
caption 分别编码，使用归一化特征内积作为相似度，并分别计算：

- 图搜文：Image-to-Text Recall@1/5/10
- 文搜图：Text-to-Image Recall@1/5/10
- RSum：六个 Recall 指标求和

## 分析与讨论

VSRN 的关键在于对 bottom-up attention 检出的区域特征建立关系图并进行图卷积推理。
因此，仅使用原始 Flickr30k 图片直接训练 CNN 编码器并不能等价复现论文设置；必须使用
与 SCAN/Bottom-Up Attention 一致的区域特征，才能保证与论文和官方代码的评测协议一致。

加分项 CLIP 与 VSRN 的差异在于：CLIP 不显式构建图结构，而是通过大规模图文对比预训练
学习统一语义空间。它适合作为较新模型的零样本检索对照，但不能替代 VSRN 的图神经网络
复现实验结论。
